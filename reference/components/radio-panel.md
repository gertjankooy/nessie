# Radio Panel

## Usage
### Use when
- Presenting a set of predefined options where only one choice is possible.
- There are two or three options.
- Option labels are short, so panels fit next to each other and save vertical space.
### Don't use when
- Users can select multiple options, or zero options.
- There are more than three options.
- The option labels are long.
- The goal is to turn an option on or off instantly — use a [Toggle](https://design.ns.nl/4a05a30ad/p/663c99-toggle).

## Placement
- Lay panel items out next to each other to save vertical space; this only works with short labels.

## Behavior
- By default, panel items share the same width to signal that the options are equivalent.

## Best practices
- Keep labels short so items can sit side by side and stay readable.

## Content guidelines
- Keep option labels short to display items next to each other.
