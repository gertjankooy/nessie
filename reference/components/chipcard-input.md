# Chipcard Input

## Usage
### Use when
- A user needs to enter their OV-chipkaart number.
### Don't use when
_Not available in ZeroHeight — to review._

## Anatomy
- **Sectioned fields**: the chipcard number split into numeric sections.

## Configurations
### Prefill
Because most OV-chipkaart numbers start with the same four digits, the input can be prefilled so users start typing from the second section. The prefilled numbers remain editable.

## Placement
_Not documented separately; placed within a form like other inputs._

## Behavior
- Focus jumps automatically to the next field as the user types.
- The keyboard changes to show numbers only.
