# Select

## Usage
### Use when
- As a last resort, when a [Stepper](https://design.ns.nl/4a05a30ad/p/37f00c-stepper), [Radio buttons](https://design.ns.nl/4a05a30ad/p/44b43f-radio-buttons), or [Radio panel](https://design.ns.nl/4a05a30ad/p/042c3f-radio-panel) won't fit.
- A user chooses one option from about 7 to 15 options and space is limited.
### Don't use when
- There are fewer than 7 options — use [Radio buttons](https://design.ns.nl/4a05a30ad/p/44b43f-radio-buttons) or [Radio panel](https://design.ns.nl/4a05a30ad/p/042c3f-radio-panel).
- There are more than 15 options — let users type into a search input with autosuggest instead.

## Behavior
- Triggers the platform's native select menu on both iOS and Android.

## Best practices
- Use a good default — when most users will or should pick a particular option, preselect it.

## Content guidelines
- Start with "Select [group name]" as a placeholder when there's no default option.
- List options alphabetically or in another logical order.
- Keep option labels brief — aim for under 5 words.
