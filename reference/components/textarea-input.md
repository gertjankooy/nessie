# Textarea Input

## Usage
### Use when
- Letting the user enter, view, and edit multi-line plain text, such as a comment in a chat or a feedback form.
### Don't use when
- Gathering short, single lines of text — use the [Text input](https://design.ns.nl/4a05a30ad/p/49d14e-input).

## Anatomy
_Shares the input anatomy (label, help text, field, error message). See [Input](input.md)._

## Configurations
### Height
The textarea defaults to roughly 104px tall and can be adjusted to the expected or allowed number of lines.

## Placement
_Not documented separately; follows [Input](input.md) form placement rules._

## Behavior
- Uses the default keyboard.
- Starts at about 104px tall; resize to the expected number of lines.
