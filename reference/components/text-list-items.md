# Text List Items

## Usage
### Use when
- Presenting simple pieces of related information that share a similar structure.
### Don't use when
_Not available in ZeroHeight — to review._

Use the Ordered type when order or sequence matters, Unordered as the default for scannable lists, and Checked to highlight benefits such as subscription discounts.

## Anatomy
- **Marker**: a number (Ordered), bullet (Unordered), or check (Checked).
- **Text**: the item content.

## Configurations
### Type
- **Ordered**: a sequence; marked with numbers by default, can be set alphabetically.
- **Unordered**: related items with no specific order; the default starting point.
- **Checked**: highlights benefits.

### Size
Available sizes are Xs, Sm, Base, and Lg.

## Placement
_Not documented separately._

## Behavior
- Ordered items are numbered by default; the order can be set alphabetically.

## Best practices
- Phrase each item consistently so users can scan; don't mix grammar or capitalization.
- Keep ordered lists between 3 and 9 items.

## Content guidelines
- Write items with roughly similar line lengths and consistent phrasing (start each with a noun or a verb).
- Avoid repeating the same word at the start of each item.
- Introduce a list with a clear heading or descriptive sentence.
- Commas and periods are allowed when items are 4 words or more.
